THRIVE - A Community-Based Goal Setting Application

THRIVE is a mobile application in which the main focus is to allow users to develop and track goals in a community-based enviornment. The application allows users to create goals and share them with friends to allow friends to motivate one another to complete these goals. This application was developed in a group of ten throughout CSE110 as our final project. You can find out more about this application at the following links:
